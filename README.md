# JobAnalyzer - 多平台岗位情况分析工具

## 🎯 功能特性

- 多平台支持（Boss直聘、智联招聘、猎聘、前程无忧、拉勾、脉脉）
- 智能数据清洗与去重
- 薪资标准化解析
- 性价比评分算法
- ECharts可视化图表
- 学历/薪资/经验多维度筛选

## 🚀 快速开始

### 方案一：纯HTML单文件（最简单）

**步骤：**
1. 访问：https://github.com/SimpleStar666/job-analyzer/blob/main/demo.html
2. **右键页面空白处** → 选择"另存为"
3. 保存为 `demo.html`
4. **双击打开**，浏览器自动显示完整界面

**特点：**
- ✅ 无需安装任何软件
- ✅ 一个文件，双击即用
- ✅ 手机电脑都能用

### 方案二：完整程序包（推荐）

**步骤：**
1. 下载：https://github.com/SimpleStar666/job-analyzer/releases/download/v1.0.0/job_analyzer_simple.zip
2. 解压到任意文件夹
3. **双击 `start.bat`**（Windows）或 `start.py`
4. 浏览器会自动打开

### 方案三：命令行方式

```bash
git clone https://github.com/SimpleStar666/job-analyzer.git
cd job-analyzer
pip install -r requirements.txt
python web/app.py
```
